# Jacky's Blog

个人博客，基于 Vite + React 19 + Tailwind CSS v4 构建。

## 技术栈

- React 19
- Vite 8
- Tailwind CSS v4
- React Router v7

## 功能

- 中英双语切换
- 亮色 / 暗色主题（默认暗色）
- 随记、项目、科研、关于页面
- 博客活跃度热力图
- 站点统计（在线天数、总字数、文章数、最后更新）

## 本地开发

```bash
npm install
npm run dev
```

## 部署

```bash
npm run build
```

将 `dist/` 目录部署到任意静态托管平台（Cloudflare Pages、Nginx 等）。
